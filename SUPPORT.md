# Support

## Where to ask questions
- Use GitHub Issues for bugs and actionable work items.
- Use Discussions (if enabled) for Q&A and design-level conversation.

## What to include (bugs)
- Repo name, commit SHA
- OS + toolchain version
- Steps to reproduce
- Expected vs actual behavior
- Logs, traces, fixtures

## Response expectations
Dust projects prioritize correctness, determinism, and spec alignment. Response times vary.
