# Dust Programming Language Organization (.github)

This repository provides **organization-level** community health files and reusable workflows for the Dust Programming Language (DPL) GitHub organization.

These files apply to all repositories in the organization unless overridden in a specific repository.

## What this repo provides

### Community health files (org-wide defaults)
- `CODE_OF_CONDUCT.md`
- `CONTRIBUTING.md`
- `SECURITY.md`
- `SUPPORT.md`
- `GOVERNANCE.md`
- Issue templates + PR template

### Reusable workflows (recommended)
- Rust CI (fmt/clippy/test)
- CodeQL scanning

See `templates/` for repo bootstrap guidance (CODEOWNERS sample, branch protection checklist, etc.).
